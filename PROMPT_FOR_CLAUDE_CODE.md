Вставь этот текст в начало задачи в Claude Code:

---

## ЗАДАЧА

Проанализируй данные CRM Bitrix24 и сгенерируй HTML-дашборд управленческой аналитики.

---

## ПРАВИЛА ОБРАБОТКИ ДАННЫХ (обязательно)

### 1. Дедупликация месяцев
Если в выгрузке один и тот же месяц встречается дважды — это дубль экспорта. Оставь только одну строку.

### 2. Аномальные суммы
Если сумма сделки > 1 000 000 000 ₽ (1 млрд) — мусорные данные. Пометить ⚠, не включать в итоги.

### 3. Воронки с нулевой выручкой
Если воронка показывает 0₽ выручки при тысячах выигранных — менеджеры не заполняют сумму. Пометить `*`.

### 4. Здоровье pipeline
Рассчитывать: (Pipeline - Просрочено) / Pipeline × 100%. Если < 10% — критический баннер.

---

## СТРУКТУРА HTML

1. Header
2. Критический баннер (если pipeline < 10%)
3. Блок замечаний по данным
4. KPI-карточки (8 штук)
5. Таблица воронок (сортировка по конверсии)
6. Динамика по месяцам
7. Проект × Месяц × Стадии (создано/закрыто) с графиками
8. Таблица менеджеров (сортировка по выручке)
9. Анализ старения
10. Воронки по стадиям (details: открыты только 2 лучших)
11. Сделки в зоне риска
12. Executive Summary
